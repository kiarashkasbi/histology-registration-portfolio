# Public Scope

This repository is a public methodology portfolio. It is not a release of MSc thesis data, private laboratory material, or unpublished results.

## Excluded Material

The repository avoids raw whole-slide images, private thesis images, sample identifiers, dataset identifiers, local file paths, real thesis overlays, private QC outputs, internal lab details, and copied private implementation code.

## Safe Material

The repository may include general explanations, workflow diagrams, synthetic examples, public-data demos with clear licensing, conceptual notes on registration and quality control, tool lists, and non-sensitive methodology documentation.

## Sharing Rule

Detailed thesis material should only be shared privately when appropriate and with supervisor or laboratory approval.
